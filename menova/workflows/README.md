# n8n Workflow Exports

Export your workflows from n8n and place the JSON files here.

## How to export:
1. Open each workflow in n8n
2. Click the three dots (...) menu in the top right
3. Click "Download" or "Export"
4. Save the JSON file with a descriptive name

- `ingestion-pipeline.json` — My workflow (Manual Trigger → Edit Fields → Pinecone)
